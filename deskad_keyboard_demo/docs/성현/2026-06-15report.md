- 로그인 화면에서 일반 앱 사이드바가 보이지 않도록 로그인 전용 스타일을 추가했다.

- Streamlit 컴포넌트가 HTML 래퍼 사이에 끼면 레이아웃이 깨질 수 있어, 최종 구조는 `st.columns`와 `st.container(border=True)` 기반으로 정리했다.
- 새로고침 시 로그인 상태가 초기화되는 문제를 수정했다.
- `origin/main` 최신 변경 사항을 현재 브랜치에 병합하고, HyperCLOVA grid_three 타임아웃/작업 큐/진행률 관련 충돌을 정리했다.
- URL query param에 인증 토큰이 노출되는 문제를 해결하기 위해 HttpOnly cookie 기반 세션 복원 구조로 변경했다.
- 로그인 성공 후 실제 토큰을 URL에 넣지 않고, 60초짜리 일회용 코드만 `/auth/cookie` 리다이렉트에 사용하도록 했다.
- FastAPI에 `/auth/session`, `/auth/cookie-code`, `/auth/cookie`, `/auth/clear-cookie` 엔드포인트를 추가했다.
- Streamlit은 `st.context.cookies`로 HttpOnly cookie를 읽고, `/auth/session` 검증을 통과한 경우에만 `session_state`를 복원하도록 했다.
